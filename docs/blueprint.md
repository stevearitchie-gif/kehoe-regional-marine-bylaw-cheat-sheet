# **App Name**: Regional Marine Bylaw Cheat Sheet

## Core Features:

- Bylaw Record Field View: Access detailed marine construction bylaw information in a mobile-first, clean card or accordion layout, showing key fields like setbacks, height limits, permit requirements, and verification status for easy on-site reference.
- Search & Filtering: Efficiently locate bylaw records by searching municipality, region, contact names, or notes, and filter results by specific region or conservation authority.
- Record Status Badges: Each bylaw record will feature clear status badges (Verified, Needs review, Missing fields, Needs source link) to quickly ascertain data reliability.
- Admin Record Management: A secure administrative interface for office staff to add new municipality records, edit existing bylaw details, and delete outdated information, including all specified fields like 'last verified date' and 'notes'.
- Bylaw Document Data Extraction Tool: An AI-powered tool within the admin interface that can analyze uploaded bylaw documents (e.g., PDFs) and suggest relevant data points (setbacks, widths, etc.) to pre-populate record fields, streamlining data entry.
- Overview Dashboard: A simple dashboard view presenting key metrics such as the total number of municipalities, total regions covered, number of verified records, and records requiring attention.

## Style Guidelines:

- A serene blue-green palette for a professional and calm feel, reminiscent of marine environments without being literal. Primary interactive elements will use a muted green-blue (#316E82), while the background offers a very light, almost greyed blue-green (#E0E9EB) for readability in varied lighting. A vibrant, fresh aqua accent (#47D1AF) will highlight calls to action and important information.
- Utilize 'Inter' (sans-serif) for both headlines and body text. Its neutral, clear, and highly legible design ensures optimal readability on mobile devices in the field and provides a modern, uncluttered look that aligns with professional use.
- Simple, clean, and functional line-art icons that complement the professional tone. Icons for search, filters, and administrative actions will be intuitively designed for quick recognition and feature large tap targets for field use. Status badges will be visually distinct using clear, bold symbols and colors.
- Mobile-first responsive design featuring generous padding and margin for readability and easy interaction. Content will be organized into digestible cards or accordions, ensuring minimal clutter and optimal information display on smaller screens. Key interactive elements like buttons will have ample size for 'large tap targets'.
- Subtle and quick transitions for navigation, search result loading, and modal interactions to provide immediate user feedback without being distracting. Animations will enhance the user experience by indicating changes in state rather than being purely decorative.
